# Travel Planner

Backend aplikacji do planowania podróży. System umożliwia użytkownikom tworzenie planów podróży, dodawanie miejsc do odwiedzenia, planowanie harmonogramu dni oraz współdzielenie planów z innymi użytkownikami.

---

# Funkcjonalności

## Zarządzanie użytkownikami

- rejestracja i logowanie użytkownika
- autoryzacja przy użyciu tokenów
- zarządzanie profilem użytkownika

## Zarządzanie podróżami

- tworzenie planów podróży
- edycja i usuwanie planów
- dodawanie opisu podróży
- określanie dat rozpoczęcia i zakończenia

## Zarządzanie miejscami

- dodawanie miejsc do odwiedzenia
- przypisywanie miejsc do konkretnych dni podróży
- zapisywanie informacji o miejscu (nazwa, opis, lokalizacja)

## Współdzielenie planów

- zapraszanie innych użytkowników do planu podróży
- role użytkowników (owner / collaborator)
- wspólna edycja planu

## Integracja z mapami

- wyszukiwanie miejsc
- pobieranie współrzędnych geograficznych
- obliczanie dystansu między punktami
